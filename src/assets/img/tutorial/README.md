# Скриншоты для туториала

Эта папка предназначена для размещения скриншотов, используемых в руководстве пользователя.

## Необходимые скриншоты:

1. **create-project.png** - Главная страница проектов и создание нового проекта
2. **start-message.png** - Создание стартового экрана чат-бота
3. **create-dialogue.png** - Интерфейс создания диалогов
4. **configure-dialogue.png** - Интерфейс наполнения диалога
5. **download-code.png** - Кнопка "Скачать код"
6. **project-files.png** - Структура файлов проекта
7. **docker-run.png** - Запуск через Docker
8. **python-run.png** - Запуск через Python

## Рекомендации по созданию скриншотов:

- Разрешение: минимум 1200px по ширине
- Формат: PNG с прозрачностью или JPG
- Качество: высокое, четкие детали
- Стиль: современный, соответствующий дизайну приложения
